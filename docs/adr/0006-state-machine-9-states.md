# 9-состояний State Machine с детерминированными переходами

**Дата:** 2026-03-09  
**Статус:** Принято  

## Контекст
Фаза 2 определила State Machine как центральный компонент Control Plane с 9 состояниями.

- **Фаза:** 2 — Control Plane
- **Компонент:** src/cryptotechnolog/core/state_machine.py
- **Класс стратегии:** SFT (управление системой)

### 9 состояний системы:
```python
class SystemState(str, Enum):
    BOOT = "boot"         # Загрузка (0-30 сек после старта)
    INIT = "init"         # Инициализация компонентов
    READY = "ready"      # Готова, но не торгует
    TRADING = "trading"  # Нормальная торговля (100% функциональность)
    DEGRADED = "degraded" # Деградированный режим (торговля ограничена)
    SURVIVAL = "survival"  # Режим выживания (только закрытие позиций)
    ERROR = "error"       # Критическая ошибка
    HALT = "halt"         # Полная остановка
    RECOVERY = "recovery" # Восстановление после ошибки
```

## Рассмотренные альтернативы
1. **Binary (Trading/Not Trading)** — слишком упрощённо
2. **3 состояния (UP/DOWN/DEGRADED)** — недостаточно для бизнес-логики
3. **12+ состояний** — избыточно, сложность управления

## Решение
Выбрана **9-состояний модель** с чёткой семантикой:

### Переходы (примеры):
| Из | Куда | Триггер |
|----|------|---------|
| TRADING | DEGRADED | RISK_VIOLATION (HIGH) |
| TRADING | SURVIVAL | RISK_VIOLATION (CRITICAL) |
| TRADING | HALT | KILL_SWITCH_TRIGGERED |
| DEGRADED | TRADING | OPERATOR_APPROVAL |
| * | HALT | EMERGENCY (любое состояние) |

### Гарантии:
- Все переходы валидируются
- Audit trail в БД
- Callbacks при входе/выходе
- Optimistic locking для concurrency

## Последствия
### Плюсы:
- **Чёткая семантика** — каждое состояние имеет определённый смысл
- **Безопасность** — невозможно перейти в недопустимое состояние
- **Аудит** — полная история переходов
- **Интеграция** — State Machine → Event Bus → другие компоненты

### Минусы:
- **Сложность** — 9 состояний требует документирования
- **Тестирование** — много комбинаций переходов
- **Operator Gate** — требуется для ручных переходов
