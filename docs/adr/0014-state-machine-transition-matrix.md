# Матрица переходов State Machine

**Дата:** 2026-03-09  
**Статус:** Принято  

## Контекст
State Machine в Фазе 2 имеет 9 состояний, но отсутствовала документация допустимых переходов между состояниями.
- Фаза 2 / Control Plane
- Класс: SFT (State Machine используется для всех типов стратегий)
- Необходимо обеспечить traceability и понимание workflow

## Рассмотренные альтернативы
1. Оставить как есть — отсутствие документации создаёт риск ошибок
2. Добавить только диаграмму в README — неполная информация
3. Добавить матрицу переходов в код с документацией — выбрано

## Решение
Добавлена матрица переходов в `src/cryptotechnolog/core/state_machine.py` с документацией всех 9 состояний:
- STOPPED, STARTING, INITIALIZING, RUNNING, PAUSING, PAUSED, RESUMING, STOPPING, ERROR

Для каждого состояния указаны допустимые переходы.

## Последствия
- **Плюсы:** Полная traceability, снижение риска ошибок, удобство для new joiners
- **Минусы:** Небольшое увеличение размера файла (документация)
