# Правила terminal UI

- `/terminal` — это рабочее место оператора, а не intro screen и не витрина будущих возможностей.
- Верхняя полоса должна читаться как единый top bar за один взгляд.
- Основная навигация должна оставаться человеческой и не смешиваться автоматически с foundation panel.
- Центральная область важнее декоративных карточек и должна забирать главное пространство экрана.
- Сначала показывать видимый рабочий результат, потом усложнять детализацию.
- Новые terminal-компоненты должны жить отдельно от foundation dashboard tree.
